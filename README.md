# Steplo

Steplo is a sleek and modern Flutter application focused on providing a beautiful authentication experience. It leverages **Material 3**, **Google Fonts**, and **Animate Do** to create a polished, animated user interface.

## 🚀 Features

* **Authentication Flow:**
    * Animated Login Screen
    * User Registration
    * Password Recovery
* **UI/UX:**
    * **Animations:** Smooth entrance animations (Fade In/Up/Down) for a dynamic feel.
    * **Typography:** Professional typography using the **Poppins** font family.
    * **Modern Design:** Minimalist aesthetic with subtle shadows, rounded corners, and abstract background shapes.
    * **Interactive Elements:** Custom input fields and seamless navigation.

## 🏁 Getting Started

Follow these instructions to set up the project locally.

### Prerequisites

* [Flutter SDK](https://docs.flutter.dev/get-started/install)
* [VS Code](https://code.visualstudio.com/) or [Android Studio](https://developer.android.com/studio)
* Git

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/steve07g/stephlo.git](https://github.com/steve07g/stephlo.git)
    cd stephlo
    ```

2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the app:**
    ```bash
    flutter run
    ```
